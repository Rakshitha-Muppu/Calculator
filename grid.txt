Golden Rule (remember this first 🧠)

Parent decides the layout
Child decides its own look

That’s it. Everything fits into this.

What to apply to the PARENT (container)

Parent controls how children are placed.

Apply layout properties here:

For GRID / FLEX (layout)

display: grid / display: flex

grid-template-columns

grid-template-rows

gap

justify-content

align-items

👉 These decide:

rows & columns

spacing

alignment

start / center / end

Example in words:
Parent says: “I will arrange you in 4 columns with equal space.”

What to apply to the CHILD (items)

Child controls how it looks.

Apply appearance properties here:

width, height

border-radius

font-size

color

background

padding

👉 These decide:

size

shape

color

text style

Example in words:
Child says: “I will be round and 40px tall.”

Calculator example (concept only)
Parent (.row)

decides → 4 buttons per row

decides → equal gaps

decides → alignment

Child (button)

decides → round shape

decides → height

decides → font size
